# Restaurant Management System

A Python + MySQL based Restaurant Management System that allows users to:
- View menu
- Place orders
- Update or cancel orders
- View previous orders
- Give feedback

## Tech Stack
- Python
- MySQL
- MySQL Connector

## Features
- Menu management
- Order booking
- Edit & cancel orders
- Feedback system
- Database connectivity

## How to Run
1. Import `database.sql` in MySQL
2. Install dependencies:
   pip install mysql-connector-python
3. Run:
   python main.py
